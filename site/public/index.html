Animating plots with gganimate and tweenr

25 Apr 2017, 19:06

Animation can be added to ggplot() plots using the gganimate package by David Robinson. Using the CSO Statbank API insert link, we can load in a table of population by county in Ireland for each Census since 1841. library(rjstat) library(tidyverse) library(gganimate) library(tweenr) library(scales) census_url <- "http://www.cso.ie/StatbankServices/StatbankServices.svc/jsonservice/responseinstance/EY001" pop_data <- fromJSONstat(census_url) pop_data <- pop_data[[1]] %>% as_data_frame() To look at county populations, filter to both sexes and filter out the total state populations.

Accessing the CSO Statbank API

22 Apr 2017, 13:44

CSO have an API for access to data on demographics and the economy in the Statbank database. The data is made available in JSON-stat format. There is an R package called rjstat , developed by Aaron Schumacher to read and write data in this format. install.packages("rjstat") library(rjstat) library(tidyverse) The website for the API is here. The JSON data is stored in a url comprising a fixed address plus the table code.
